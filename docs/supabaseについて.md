# Supabase について



## 要約

| 観点 | 評価 |
|------|------|
| **コスト** | 月額約3,800円で、自社構築と比較して大幅なコスト削減を実現 |
| **利用制限** | 現在の利用規模では上限に余裕あり。超過時も事前通知があり、予期せぬ費用は発生しない |
| **安全性** | 国際的なセキュリティ認証を取得。通信・保管・アクセス制御の三方面から多層防御を実施 |



## 1. Supabaseとは

Supabase は、Webアプリのデータ管理・ユーザー認証・ファイル保存などの機能をまとめて提供するクラウドサービスです。  
世界中の企業・スタートアップで幅広く採用されており、信頼性の高いサービスとして実績があります。



## 2. 費用について

現在、無料プランで開発中です。  
本稼働時は**月額$25（約3,800円）のProプラン**を利用する予定です。

### Proプランで利用できる主な内容

| 項目 | 内容 |
|------|------|
| データ保存容量 | 8GBまで（超過分は従量制） |
| バックアップ | 毎日自動でバックアップを取得 |
| サポート | メールサポートあり |
| セキュリティ | 本番運用向けの安全対策を標準提供 |

### 自社で同等の環境を構築した場合との比較

同じ機能（データ管理・ログイン認証・ファイル保存）をAWSなどのクラウドで自前構築した場合と比較すると、以下のコスト削減効果があります。

- 初期の構築・設定が不要
- サーバー管理・メンテナンスの作業が不要



## 3. Proプランの利用制限について

Proプランには月ごとの利用上限が設定されています。上限を超えた場合は、超過分のみ追加費用が発生します。

### 利用上限と超過時の料金

| 項目 | 月間上限 | 上限超過時の追加料金 |
|------|----------|----------------------|
| ファイル保存容量 | 100GB | 1GBあたり約3円 |
| 月間利用ユーザー数 | 100,000人 | 1,000人あたり約500円 |

### 費用超過を防ぐ仕組み

Proプランでは、**デフォルトで「支出上限」が有効**になっています。これにより、上限を超えた場合にサービスが一時停止されることはありますが、**予期せず多額の費用が発生することはありません**。



## 4. 安全性について

Supabaseは、企業の本番サービスとして利用されることを前提に設計されており、**多層的な安全対策**を標準で提供しています。


### ① データの保管場所

- データは **Amazon（AWS）のサーバー**に保管されており、世界最高水準のデータセンターを利用しています


### ② 通信・保管データの保護

- インターネット上を行き来するすべてのデータは、**外部から読み取れない形（暗号化）** で送受信されます
- サーバーに保管されているデータも同様に暗号化されており、万が一サーバーに不正侵入があっても内容を読み取ることができません



### ③ 不正アクセスへの対策

- **ログイン認証の強化**：管理者アカウントには二段階認証を設定しており、パスワードが漏れた場合でも不正ログインを防止できます
- **アクセス権限の制限**：ユーザーは自分に関係するデータにしかアクセスできない仕組みを、システム側で強制的に適用しています
- **大量アクセス攻撃への対策**：外部からの悪意ある大量アクセス（サイバー攻撃）を自動的に検知・遮断する機能を内蔵しています
- **アクセス記録の保存**：誰がいつ何にアクセスしたかのログを記録しており、不審な操作があった場合に追跡が可能です


### ④ 第三者機関による安全性の認証

Supabaseは、外部の専門機関による審査を受け、以下の国際的なセキュリティ認証を取得しています。

| 認証・規格 | 内容 |
|------------|------|
| **SOC 2 Type II** | 情報セキュリティ管理の仕組みが適切かを第三者が審査・認定する国際資格 |
| **GDPR準拠** | EUの厳格な個人情報保護ルールに対応済み |

これらは、銀行や大手企業のシステムにも採用される水準の認証です。
